# Example: SaaS Product Brand

Sample run of `brand-launchpad` on a fictional SaaS for indie developers
called **Loomstack** (project management for solo founders).

> All names, numbers, and outputs are illustrative only.

---

## Step 0: Capture

```yaml
project_name: loomstack
brand_type: product
real_name_or_alias: "@loomstack"
one_sentence_what_it_does: "Project management built for solo founders shipping multiple products."
icp:
  who: Indie hackers running 2-5 side projects
  pain_or_desire: Notion is too heavy, Linear is overkill, Trello is too dumb
  where_they_hang_out: Indie Hackers, X (build-in-public crowd), YouTube
desired_outcome_12mo: 500 paid users at $19/mo
constraints:
  anonymous_required: no
  time_per_week_hours: 6
  camera_comfort: reluctant
  platforms_committed: [X, YouTube, Indie Hackers]
```

## Step 3: Positioning sentence

> Loomstack is the only project tool for indie founders that scales down
> as your project list grows.

(Note the inversion: most PM tools brag about "scaling up". Counterposition
the dominant frame.)

## Step 5: Hero line candidates

| Candidate | Visualize | Falsify | Nobody else |
|-----------|-----------|---------|-------------|
| "The only PM tool that fits 5 projects on one screen." | Pass | Pass | Pass |
| "Lightweight project management for indie hackers." | Fail | Fail | Fail |
| "Notion was built for teams. Loomstack was built for one." | Pass | Pass | Pass |

Winner: candidate 3. Falsifiable (Notion's origin is teams), visualizable
(reader sees the solo developer), and nobody else can claim it.

## Step 6c: Named recurring series

1. **"Built solo"** — Wednesday Reels showing one feature shipped without
   a team. Real commits only.
2. **"$X/mo to run a project"** — Biweekly Tuesday breakdown of actual
   AWS / Vercel / database bills for a Loomstack user's side project.
3. **"PM tools that broke me"** — Monthly Friday teardown of a previous
   tool the founder abandoned and why.

## Step 9: Day-21 milestone

Get one indie hacker with 3+ active side projects to migrate from Notion
or Linear by Day 21. Capture the before/after screenshots with permission.
Week 4 content is 100% sourced from this migration.
